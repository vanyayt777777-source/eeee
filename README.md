# Telegram Bot «Конструктор магазинов»

Telegram бот для создания и управления интернет-магазином с интеграцией Crypto Bot для приема платежей в USDT.

## Возможности

- 🛒 **Каталог товаров** с категориями и описаниями
- 💱 **Мультивалютность** (рубли и доллары)
- 💳 **Прием платежей** через Crypto Bot (USDT)
- 👤 **Личный кабинет** пользователя с историей покупок
- 🔧 **Админ-панель** для управления магазином
- 📊 **Статистика** продаж и пользователей
- 📨 **Рассылки** всем пользователям
- 📁 **Управление категориями и товарами**

## Технологии

- Python 3.10+
- aiogram 3.x
- SQLite
- aiohttp
- Crypto Bot API
